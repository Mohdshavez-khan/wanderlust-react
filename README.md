# Wanderlust
A full-stack MERN travel listing application where users can explore create, and manage travel listings.

## Features
- User Authentication (JWT) and Role-based Authorization.
- Create, Edit and Delete Listings.
- Image Upload (cloudinary)
- Interactive Maps (Mapbox)
- Search and Filters
- Responsive Design
- Loading Spinner
- Form Validation
- Error Handling
- Category Filters

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- Bootstrap
- JWT
- Cloudinary
- Mapbox
- Multer
- Joi validation

- ## Run Locally
  ```bash
  npm install
  npm run dev
  ```
